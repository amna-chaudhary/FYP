Help & Support

🔹 1. How to Use Chatbot

You can interact with the system using natural language.

Examples:
• "What is a Green Energy Certificate?"
• "Create 50 solar certificates in Lahore"
• "List certificate 1 price 100"

🔹 2. How to Generate Certificate

Step 1: Type your request in chat
Step 2: Example → "issue 10 solar certificates location Karachi"
Step 3: Certificate will be generated and displayed

🔹 3. How to Use Marketplace

• List: "list cert 1 price 100"
• View: "show marketplace"
• Buy: "buy cert 1"

🔹 4. Supported Commands (VERY IMPORTANT for demo)

• issue [amount] [type] location [city]
• list cert [id] price [amount]
• buy cert [id]

🔹 5. Troubleshooting

• Make sure backend servers are running
• Check internet connection
• Try simple commands if AI doesn't understand

About

🔹 Project Name & Description

Green Energy Certificate (GEC) System

A blockchain-based system that allows users to generate, manage, and trade green energy certificates using AI-powered interaction and decentralized technology.

🔹 Key Features

• AI Chatbot for natural interaction
• Certificate generation (Solar, Wind, Hydro)
• Blockchain integration (Aptos)
• Marketplace for listing certificates
• RAG-based knowledge assistant
